Common Tools

GDB

Invoking GDB

gdb executable ;start with an executable program.
gdb -e executable -c core-file ;start with both an exec and core.

The commonds in the GDB

show information:

help [topic | comand] ;list the help about commond.

info args ;list the args in the commond line;
info locals ;list all the local variables;
bt full ;list all variables;
info breakpoints/br [n] ;list all breakpoint or br numberd n;
info watchpoints ;list all watchpoints;

where ;list the current line number and function.

Breakpoints and Watchpoints

watch cond-expr ;set a watchpoint when meeting condition.
rwatch expr ;interrupt when reading expr;
awatch expr ;break when ether reading or modifing expr;

clear [function name|line number] ;delete breakpoint
delete(d) [breakpoint number|breakpoint range] ;delete breakpoint numberd no.
disable [breakpoint number|breakpoint range]
enable [breakpoint number|breakpoint range]

Execute By One Step

step(s) [n] ;execute next line code and will go into the function;
next(n) [n] ;execute next line code and will not go into the function;
stepi(si) [n] ;execute the next assemble or CPU instruction;
nexti(ni) [n]

until run until the certain line number, address or function is meet.
until line-number
until line-number if condition

return
return expression pop the selected stack frame. return the value .

finish the function will be finished.

Stack of Program

frame(f) number ;select the certain stack frame;
up [number] ;select the upper stack frame based on the current stack frame;
down [number] ;select the down stack frame based on the current;

bt full ;print all the local variables in all the stack frames;
info frame [addr] ;describe the selected stack frame;
info args ;describe the selected parameteres, local variables and exception functions.
info locals
