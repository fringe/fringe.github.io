leveldb源码剖析（002）-slice

学习知识点：

1.使用直接初始化，而不是复制初始化，加快程序运行速度；
2.对于任意的非内置的数据，全部使用指针或引用；
3.对任意的char操作都不涉及分配内存和回收内存的操作；
4.intmemcmp(const void buf1, const void *buf2, unsigned int count); 比较内存区域buf1和buf2的前count个字节。该函数是按字节比较的。
